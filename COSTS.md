# Launch Cost Breakdown

## Minimum Required Budget

### Essential Costs (Must Have)

1. **Token Creation on Pump.fun**
   - Cost: **~0.1-0.5 SOL** (~$15-75)
   - Includes: Token mint, bonding curve setup
   - Note: Pump.fun has very low fees, usually just network fees

2. **Streamflow Vesting Lock**
   - Cost: **~0.117 SOL (~$16-20 USD)** at current SOL prices
   - Includes: 10-year vesting lock for 200M tokens (20% supply)
   - Optional auto-claim: 0.19 SOL (~$26-30) for automatic token distribution
   - Transaction fees: Minimal Solana network fees
   - Note: Streamflow charges 0% fees on tokens distributed, only network fees for contract creation

3. **Initial Token Investment**
   - Cost: **€1000** (as per your commitment)
   - Purpose: Initial buy-in to show commitment

4. **DEX Upgrade (Dexscreener)**
   - Cost: **$299**
   - Includes: Premium chart features, social links
   - Note: Can be paid from creator fees after launch if needed

### Optional but Recommended

5. **Shop Setup**
   - Cost: **$0-500**
   - Free options: Printful + Stripe (pay per transaction)
   - Paid options: Shopify (~$29/month + transaction fees)
   - Recommendation: Start with Printful + Stripe (free setup)

6. **NFT Minting**
   - Cost: **~$5-50 per NFT** (Solana fees are very low)
   - Includes: Minting first few quote artwork NFTs
   - Note: Can be done gradually, not needed at launch

7. **Website/Infrastructure**
   - Cost: **$0** (already have website)
   - Hosting: Already set up
   - Note: Just your time investment

## Total Minimum Budget

### At Launch (Essential):
- Token creation: **~$15-75** (0.1-0.5 SOL)

- Initial investment: **€1000** (~$1080 USD)
- **Total: ~$1,111 - $1,175 USD** (without auto-claim)
- **Total: ~$1,137 - $1,205 USD** (with auto-claim recommended)

### Within First Week:
- DEX upgrade: **$299**
- **Additional: $299**

### Within First Month:
- Shop setup: **$0-500** (optional)
- NFT minting: **$50-200** (optional)
- **Additional: $50-700** (optional)

## Realistic Total Budget

**Minimum to launch: ~$1,111 - $1,175 USD** (without auto-claim)
**Recommended: ~$1,137 - $1,205 USD** (with auto-claim for better UX)

This includes:
- ✅ Token creation and launch
- ✅ 10-year vesting lock setup
- ✅ Initial €1000 investment
- ✅ DEX upgrade

**With shop and NFTs: ~$1,500 - $2,000 USD**

## Cost Optimization Tips

1. **Streamflow Lock**: ~0.117 SOL (~$16-20) for vesting contract creation. Consider adding auto-claim (0.19 SOL) for better user experience - tokens automatically unlock without manual claiming.
2. **Shop**: Start with free Printful integration, upgrade later if needed
3. **NFTs**: Mint gradually as you generate revenue
4. **DEX Upgrade**: Can wait until after launch when you have creator fees

- Solana transaction fees are very low (typically < $0.01 per transaction)
